# SOP: Suspicious File Path Simulation

## Purpose

Test detections based on unusual file locations.

## Steps

1. Create a text file:

```
test.txt
```

2. Place it in:

```
C:\Users\Public\Music\
```

3. Optionally modify timestamps.
4. Review:

* USN Journal
* Velociraptor filesystem artifacts
* SIEM alerts

## Notes

* Safe for all environments.
