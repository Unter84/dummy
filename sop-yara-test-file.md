# SOP: YARA Trigger File

## Purpose

Test YARA scanning without involving malicious content.

## Steps

1. Create a text file:

```
This file contains the word malware for testing.
```

2. Save it as:

```
yara-test.txt
```

3. Run YARA rules over the file.
4. Review alerts in your detection tools.

## Notes

* The file is harmless.
